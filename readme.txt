Export: This is a inside service, this meand, none of information send to other url. In this servies we generate a CSV file containing information about selected products.
Filter:  This is a inside service too that using for search products and filter them. The input parameters are including An array of information entered in the filter form and response An array of search results: Max_num_pages, Found_posts, Product_ids and Variation_ids
